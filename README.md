# نظام إدارة مشاريع البناء (إدارة الموارد، التكلفة، الجدول الزمني)
==========================

## ملخص المشروع
----------------

نظام إدارة مشاريع البناء هو تطبيق مفتوح المصدر يهدف إلى إدارة الموارد، التكلفة، والجدول الزمني للمشاريع البناءية. يهدف هذا النظام إلى تسهيل إدارة المشاريع البناءية من خلال توفير أدوات متكاملة لمراقبة الموارد، التكلفة، والجدول الزمني.

## هيكل المشروع
----------------

### مكونات المشروع

* `backend`: يحتوي على الكود المصدر للبنية التحتية (API، قاعدة البيانات، إلخ).
* `frontend`: يحتوي على الكود المصدر للواجهة المستخدمة (تطبيق الويب، تطبيق الهاتف المحمول، إلخ).
* `docker`: يحتوي على الملفات اللازمة لتشغيل النظام باستخدام Docker.

## تشغيل النظام باستخدام Docker
--------------------------------

### خطوات تشغيل النظام

1. تأكد من أن Docker و Docker Compose مثبتين على جهازك.
2. انتقل إلى مجلد المشروع وexecute الأمر التالي:

bash
docker-compose up


3. بعد تشغيل النظام، يمكنك الوصول إلى الواجهة المستخدمة من خلال الرابط `http://localhost:8000`.
